# PRIPSH-INTERNSHIP: SassySnippet Manager

## Project Overview
This project is a professional-grade **Full-Stack Code Snippet Manager** developed as part of the PRIPSH-INTERNSHIP program. It allows users to store, manage, and share code snippets within a secure, high-performance environment.

## Features
- **Team Workspaces**: Create and manage shared vaults for team collaboration.
- **Advanced Editor**: Real-time code linting and syntax-aware interface.
- **Bulk Data Management**: Support for exporting snippet libraries as JSON and importing via .ZIP files.
- **Security**: Robust user authentication with password hashing using Werkzeug.
- **Smart Organization**: Library filtering, favorites system, and a "Soft Delete" trash bin for recovery.

## Tech Stack
- **Backend**: Python / Flask
- **Database**: SQLite3 (Relational Storage)
- **Frontend**: Tailwind CSS / JavaScript (Reactive UI)
- **Security**: PBKDF2 Password Hashing
